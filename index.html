WYSIWYG vs Hand coding

Website builders like squarespace, wix, and cargo can be identified as What You See Is What You Get (WYSIWYG). This describes any building program with a graphical interface and tools like drag/drop that let you visually place element as they would be displayed to the end user - microsoft word is another example.
Hand coding a webpage using HyperText Markup Language (HTML) and Cascading Style Sheets (CSS) is a manual process of development, which allows nearly-complete customisation of the enire page.
